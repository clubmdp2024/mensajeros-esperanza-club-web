REGISTRO FOTOGRÁFICO

Evidencias visuales organizadas por actividad.

El sistema de carrusel está pensado para escritorio y permite que cada actividad tenga una cantidad diferente de fotos, sin necesidad de duplicar mucho código.

Galería por actividades

Esta página genera los carruseles desde JavaScript para que sea más fácil administrar actividades con distinta cantidad de fotografías. Si una actividad tiene 2 fotos, 5 o 12, solo debes cambiar el arreglo galleryData dentro de /js/main.js.
