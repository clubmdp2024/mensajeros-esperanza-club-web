# Carpeta digital del club

## Estructura
- index.html -> portada principal
- styles.css -> estilos exclusivos del inicio
- /pages -> subpáginas del sitio
- /styles/pages.css -> estilos compartidos de subpáginas
- /js/main.js -> menú activo y carruseles del registro fotográfico
- /pdf -> PDFs enlazados
- /img -> imágenes raster
- /svg -> logos vectoriales de reemplazo

## Cómo editar rápido
1. Reemplaza los PDFs en /pdf conservando los nombres:
   - hojas_de_vida.pdf
   - cronograma.pdf
2. Cambia los textos de historia en /pages/historia.html
3. Ajusta visión, misión y propósito en /pages/vision-mision-proposito.html
4. Agrega fotos por actividad desde /js/main.js y guarda las imágenes en /img/registro/
